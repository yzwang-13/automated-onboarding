# README

## Run

1. Install python dependencies
   ```sh
    pip3 install -r ./requirements.txt
   ```

2. Run this project
   ```sh
    python3 csv-client.py
   ```

3. Dockerize this project
   ```sh
    docker build -t automated-onboarding .
   ```
   run in interactive mode

   ```sh
    docker run -it automated-onboarding
   ```
